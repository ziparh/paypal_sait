Этот проект представляет собой веб-приложение на Flask, которое позволяет пользователям делать донаты через PayPal на борьбу с глобальным потеплением и загружать изображения для анализа мусора с помощью искусственного интеллекта. Пользователь может указать сумму пожертвования и выбрать процент на поддержку создателя. Все транзакции обрабатываются через PayPal API и сохраняются в базе данных SQLite.

Основные возможности:

Пожертвования через PayPal с возможностью выбора процента поддержки создателя.
Анализ мусора с помощью ИИ: распознаёт объекты мусора (бутылки, упаковки, отходы) на загруженных изображениях.
Запись транзакций в базу данных и удаление исходных изображений после анализа.

Приложение удобно для экологических инициатив и сбора средств, сочетая финансовую поддержку с борьбой за чистую окружающую среду.
